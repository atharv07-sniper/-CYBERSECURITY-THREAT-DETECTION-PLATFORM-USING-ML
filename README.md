# CYBERSECURITY-THREAT-DETECTION-PLATFORM-USING-ML
This project was created as a smart cyber-security threat detection system with AI capabilities to observe live traffic on networks and detect suspicious network traffic immediately. The system passes network packets through Wireshark via the TShark API, which subsequently analyzes the behavior of the packets, and classifies them as SAFE or THREAT based on the behaviour, using a Random Forest Machine Learning model. The project also features an interactive real-time visualization dashboard using OpenCV that presents the live monitoring information and threat and activity count of network traffic.

The primary objective of this project is to prove the feasibility of creating an adaptive, intelligent and versatile Intrusion Detection System (IDS) that is able to react to emerging cyber threats, which are not directly covered by its rule-base, using Machine Learning.
